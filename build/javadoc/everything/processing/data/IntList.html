Class IntList

  • java.lang.Object
    • processing.data.IntList
  • All Implemented Interfaces:
    java.lang.Iterable<java.lang.Integer>

    public class IntList
extends java.lang.Object
implements java.lang.Iterable<java.lang.Integer>
    Helper class for a list of ints. By design (for efficiency), functions like sort() and shuffle() always act on the list itself. To get a sorted copy, use list.copy().sort().
